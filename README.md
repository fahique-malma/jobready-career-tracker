# JobReady - Career Tracker

A client-side responsive Job Application Tracking System built using HTML, CSS, and JavaScript.
JobReady helps users organize their job search by tracking applications, interview stages, offers, rejections, deadlines, and notes—all from a clean dashboard interface.

## ✨ Features

* Dashboard with real-time statistics
* Add new job applications
* Edit existing records
* Delete job applications
* Track application status - Saved, Applied, Interview, Offer, Rejected
* Store data using Local Storage
* Responsive design for desktop, tablet, and mobile devices
* Recent Applications section
* Clean and modern UI
